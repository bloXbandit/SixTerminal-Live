# LAWS
1. Nothing grades its own homework — four parties per tick, the last one deterministic.
2. One item per tick. The conductor picks ONE.
3. The verifier is fresh and sees spec + diff only. The maker's confidence is not evidence.
4. Never iterate on a swapped model — a stuck worker is a reroute for the next tick.
5. Autonomy is per skill, not per loop. Any fail demotes to watch.
6. Finishing IS the enrollment — a goal enrolls when its predicate first passes.
7. The ratchet number NEVER goes up in a surviving commit. Never game the metric.
8. Nothing lands unsigned — compost proposals require the human's signature.
9. Flaky predicates → quarantine, never delete.
